LayerElementsObjectLoader

Generates a single random map using individual element JSON files. Each element (house, tree, etc.) is defined in its own JSON file specifying the tile layers and their contents.

Best for: projects where each element type has its own separate tile definitions file.

Generator Data example:
map-data.json

Expected JSON files:
house-001.json
house-002.json
tree.json

Expected Images:
tilesheet.png

All JSON files in the examples use the same tileset image. In theory it works with individual images per element but this still needs testing - use the composite option for multiple tileset images.

LayerElementsCompositeLoader

Generates a single random map using a composite JSON file that bundles all element definitions and multiple tileset images together. The generator merges tilesets automatically using the Tile Map Optimizer.

Best for: projects using the Reldens tileset analyzer output, or when all elements share a single optimized tileset.

Generator Data example:
map-composite-data.json

Expected JSON files:
reldens-town-composite.json

Expected Images:
outside.png
terrain.png
house.png

Single tile map JSON with multiple tilesets and images. Original contents are 16x16 - set "factor: 2" to run the optimizer and get 32x32 tiles.

MultipleByLoaderGenerator

Generates multiple independent random maps from a single composite file. Each map is generated separately using the same element definitions but different random layouts.

Best for: generating a set of unique maps (e.g., multiple town maps) in a single operation.

Generator Data example:
map-composite-data-with-names.json

Expected JSON files:
reldens-town-composite.json

Same composite structure as SINGLE but with multiple map names - uses a different loader to create all maps at once.

MultipleWithAssociationsByLoaderGenerator

Generates multiple maps with defined associations between them - for example, a town map and its connected house interiors. Sub-maps (interiors) are generated with their own configuration and linked to entries in the main map.

Best for: Reldens game worlds where players can enter buildings or transition between connected areas.

Generator Data example:
map-composite-data-with-associations.json

Expected JSON files:
reldens-town-composite-with-associations.json
house-composite.json

Multiple maps with association options - uses a different loader to create all associated maps at once.
