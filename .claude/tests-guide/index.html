E2E Test Verification Guide

Each page documents one test case with step-by-step actions, code references, and proof links.

  • Double Login Disconnects First Session

Chat

Clans

Combat

  • Player Dies From Enemy And Revives After Timeout
  • Combat Skill (Attack) - attackShort Deals Damage In Chat
  • Combat Skill (Physical Attack) - attackBullet / fireball Deals Damage In Chat
  • Combat Skill (Effect) - heal Applies Effect On Cast
